Quest Log

Main research lanes

Same core content as the original site, but reorganized into a clearer map of the work: what I study, where I have worked, and which outputs best represent that path.

01 / Systems

Post-training LLMs

I focus on what happens after pretraining: optimization choices, reward design, and evaluation loops that change how a model behaves in practice.

02 / Control

Reinforcement fine-tuning

I am especially interested in reinforcement-based post-training methods that improve alignment, robustness, and generalization without sacrificing useful behavior.

03 / Code

AI for code

Coding models are a recurring target domain in my work, from reliable problem solving to long-horizon software evolution settings.

04 / Cognition

Reasoning tasks

I care about models that stay coherent under multi-step reasoning pressure, not only on short benchmark prompts but also on harder workflows.

05 / Safety

Reliability and alignment

The through-line is dependable behavior: systems that are easier to steer, easier to trust, and easier to evaluate when the environment is messy.
